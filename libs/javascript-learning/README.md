# javascript-learning

Plain-JavaScript concept & interview practice — a pub/sub event emitter plus a
notes file of exercises (`this`/`call`/`apply`/`bind`, hoisting, closures, array
flatten, deep-get by path, the circles-colour puzzle, …).

**Not wired into the app** — nothing imports this lib; it's a kept sandbox of
learning snippets. So it deliberately has **only a `lint` target** (no
`typecheck`, no barrel/alias, no page) — unlike the feature libs, which are
importable via `@my-portfolio/<name>`.

## Targets

```bash
pnpm nx lint javascript-learning
```
