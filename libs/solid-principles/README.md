# solid-principles

A tiny sandbox demonstrating the Single Responsibility Principle (a calorie
`Tracker` that delegates logging to an external function). Authored in TypeScript.

```tsx
import SolidPrinciples from '@my-portfolio/solid-principles';
```

Rendered by [pages/solid-principles.jsx](../../pages/solid-principles.jsx).

```bash
pnpm nx lint solid-principles
pnpm nx typecheck solid-principles
```
